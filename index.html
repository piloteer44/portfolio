Projects

Santos Pizza

Website for a fictive pizzeria called "Pizzeria Santos". This website was the first project that me and three other students got to develop during the start of our fourth technical year at NTI-Gymnasiet. The website was created using HTML and CSS together with Javascript. Using Javascript we created features such as a dynamic closed days schedule that changes depending on which calender day is closest, and a menu in which you can search whether or not the pizzeria delivers pizza to your postal area. We developed a program that takes screenshots of the website in different resolutions to test the design on the different resolutions. We also made tests for the website using Selenium for Python. These tests tested the different functions on the website such as the navbar buttons and the delivery check system.


Twitter API live feed search

A tool developed for searching through the twitter live feed based using Twitters API. The tool pulls the tweet from the live feed depending on what parameter(s) the user gives it, these parameters could be specific words, hashtags and/or if the tweet has a image or video and more.


Lunch NFC system

A system developed for the lunch cafeteria at NTI-Gymnasiet during the Covid-19 pandemic. It's a system for scanning a personal tag when entering the cafeteria, it then checks whether or not the individual is on time for their lunch. If the individual is on time, a display shows "Welcome", and if the person is not on time or not in the tag system it shows a display for the respective response. The system was programmed in Python and uses a Raspberry Pi that runs the program. The Raspberry Pi is then connected to a screen that displays the response from the program.
