Projects

Santos Pizza

Website for a fictive pizzeria called "Pizzeria Santos". This website was the first project that me and three other students got to develop during the start of our fourth technical year at NTI-Gymnasiet. The website has advanced features such as a dynamic closed days schedule that changes depending on which calender day is closest, and a menu in which you can search whether or not the pizzeria delivers pizza to your postal area.


Twitter API live feed search

A tool developed for searching through the twitter live feed based. The tool pulls the tweet from the live feed depending on what parameter(s) the user gives it, these parameters could be specific words, hashtags and/or if the tweet has a image or video and more.


Lunch NFC system

A system developed for the lunch cafeteria at NTI-Gymnasiet during the Covid-19 pandemic. It's a system for scanning a personal tag when you enter the cafeteria, it then checks whether or not you're in time for your lunch. If the individual is in time, a display shows "Welcome", and if the person is not on time or not in the tag system it shows a display for the respective response.
